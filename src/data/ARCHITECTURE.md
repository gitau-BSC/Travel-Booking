# Travellite Architecture

## System Overview
Travellite is a modern bus booking system built with React and Tailwind CSS. It follows a component-based architecture with clear separation of concerns.

## Tech Stack
- **Frontend**: React 18, Next.js 14
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Build Tool**: Next.js
- **Package Manager**: npm
